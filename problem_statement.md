# Project: COURTROOM AI, A Multi Agent Legal Debate System

## Goal
For education purpose of law Students 

## Problem Statement
The problem this project solves.

## Key Features
1.Interactive Rounds: Step through the trial round-by-round.
2.Automatic Verdict: The Judge automatically halts the debate if sufficient evidence is gathered.
3.Real-time Fact-Checking: The Judge uses Tavily (web search) to verify claims made by both sides.
4.Strategic Depth: Strategist agents analyze the opponent's case to brief the advocates before they speak.
5.Role-Specific Personas: Agents adhere to strict legal personas, terminology, and constraints.

## Technical Stack
- Frontend: Streamlib
- Backend: Python code
